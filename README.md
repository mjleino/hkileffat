HKILEFFAT
=========

## Requirements

- Python 2.6 or 2.7
- [pystache](https://github.com/defunkt/pystache)

## Usage

1. [grab an API key](http://api.omakaupunki.fi/)
2. paste it into `leffat.conf`
3. run `python grab-showtimes.py output.html`
